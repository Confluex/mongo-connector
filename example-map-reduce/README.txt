HOW TO DEMO:
  1. Set the following environment variables:
    a. mongoUsername This is the username of your Mongo Account
    b. mongoPassword This is the password of your Mongo account
  	c. mongoAddress This is the url of your Mongo server
  2. Run the different setup flows from the MongoFunctionalTestDriver, or deploy it an a Mule Container:
  	a. CreateProductsFlow: Creates some price update document in the mongodb. Alternatively, hit
  	http://localhost:9090/mongo-demo-setup-create-price-updates
